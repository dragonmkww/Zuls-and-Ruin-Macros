# Zuls and Ruin Macros

Just download the zip files from releases to use them. Yes, these are the official latest versions.

You can scan every single file with Virustotal if u want, none of them are viruses.

For Zuls macro if u dont have a license key then use the AHK scripts and change the keybinds using VSCode, it's the same thing just without a GUI.

If there's any problems using them make an issue.

Enjoy!
